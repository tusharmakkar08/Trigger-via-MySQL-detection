Trigger For MySQL Insert 
========================

A demo for pushing on Redis whenever an Insert operation is done on MySQL

Language Used :
--------------

* Python
* Shell Scripting

Database Used :
---------------

* MySQL
* Redis

Libraries Used :
----------------
	
* redis
* MySQLdb
* sys
* time
* random

How to Run :
-----------
<pre>
				sh automaticscript.sh	
				# Run Script for creating database and table
				python trigger.py		
				# Running python code
</pre>
